# Tic Tac Toe Game 🎮

A simple console-based Tic Tac Toe game written in C++.

## Features
- Two player game (X and O)
- Interactive console board
- Win detection (rows, columns, diagonals)
- Draw detection
- Menu system with instructions

## Concepts Used
- Arrays (2D board)
- Functions
- Loops
- Conditional statements
